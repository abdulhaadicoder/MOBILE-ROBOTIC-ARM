# MOBILE-ROBOTIC-ARM
A mobile robotic arm mounted on 4 wheels, controlled using Ir REMOTE and powered by Arduino Uno. Fully maneuverable with precise arm control.
Component	              Arduino Pin
IR Receiver OUT                	D2
Servo (base)                    D5
Servo (middle)                  D6
Servo (for claw)              	D7
Green LED (+)	                  D8
LED GND → Resistor → GND      	—
